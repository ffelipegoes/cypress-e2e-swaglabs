## Resumo do Projeto

Este projeto consiste na automação de testes end-to-end utilizando o Cypress para o site Swag Labs.

Os testes foram desenvolvidos com foco na validação dos principais fluxos da aplicação, como autenticação, manipulação de produtos e finalização de compras. A estrutura foi organizada de forma modular, separando os cenários de teste por funcionalidades e os comandos customizados por domínio, visando melhor escalabilidade e manutenção.

---

## Boas Práticas Aplicadas

Foram implementadas boas práticas como:

- Separação de testes por feature (Login, Checkout, Produtos, Logout)
- Criação de custom commands reutilizáveis
- Organização dos comandos em múltiplos arquivos
- Uso de dados dinâmicos (faker) para simulação de usuários
- Estrutura preparada para crescimento e fácil manutenção

---

## Cobertura de Testes

Os cenários cobrem tanto fluxos positivos quanto negativos, garantindo maior confiabilidade na aplicação.
